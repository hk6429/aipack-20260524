第二課：思緒整理與形成觀點

PR 法（三個理由製造機）
  • Point：用 1 句話敘述觀點（「我的看法是…」「我的觀點是…」）
  • Reason：提出 3 個理由：
    1. 主要的原因是…（事實）
    2. 而且…（邏輯脈絡）
    3. 除此之外…（感覺與感受）
重要的事情，用三種方法講三次（各一次）。
KJ 法（整理雜亂訊息）
  1. 整理：不批評，想到就寫
  2. 分類：3–4 個項目為佳
  3. 排序：定小標 ＋ 一句話說明
  4. 主題：金句化、好記易流傳

第三課：邏輯表達的 PREPA 架構

🔥為觀點下標題
🔥簡潔的說明
🔥佐證、舉例、說故事
PREPA 五步驟完整版
  1. ① 用一句話陳述觀點（我認為…）
    • Ⓐ Point — 用 1 句話敘述觀點（我的看法是 / 我的觀點是）
    • Ⓑ Reason — 提出 3 個理由說明
      1. 主要的原因是（事實）
      2. 而且（邏輯脈絡）
      3. 除此之外（感覺與感受）
  2. ② 提出原因、理由說明（根據…）
    • ❶ 合乎主題意義
    • ❷ 貼近聽者經驗
    • ➌ 真實故事
  3. ③ 舉個例子來說（例如…）
  4. ④ 再次重申觀點（所以我認為…）
  5. ⑤ 呼籲行動（請大家…）
    • ❶ 簡單化：計畫越簡單越容易推動
    • ❷ 微改變：在既有習慣中改變

第四課：以人為本的 4F 表達策略

1️⃣ Fact

根據事實

2️⃣ Feeling

變得更好

3️⃣ Finding

連結經驗

4️⃣ Future

解決問題

1️⃣ 根據事實 Fact（5W1H 自我檢核）

  • Who：你的身分是否適合說這些話？→ 酸民？專家？
  • What：如果是意見，有足夠根據嗎？如果是事實，是否有所保留？→ 意見？事實？
  • When：什麼時候是最適當的表達時間？→ 事前？事後？
  • Where：哪裡是最適合的表達地點？→ 公開？私下？
  • How：你表達時候的態度如何？→ 理性？感性？
  • Why：你是否有立場？是否刻意在美化或醜化？→ 善意？惡意？

2️⃣ 變得更好 Feeling（四感建立）

  1. 存在感
  2. 意義感
  3. 價值感
  4. 希望感

3️⃣ 連結經驗 Finding（六個內在維度）

  1. 感受：身體的感受、情緒的感受
  2. 情緒後的決定：面對、逃避
  3. 期待：對自己的、對他人的、來自他人的
  4. 觀點：概念、經驗、成見、信念、假設、預設立場、主觀現實、認知
  5. 渴望：被愛、可愛的、被接納、被認同、有意義、有價值
  6. 自我：生命力、精神、靈性、核心、本質

4️⃣ 解決問題 Future（行動四階）

  1. 願景 Vision
  2. 目標 Goal
  3. 任務 Mission
  4. 開始行動 Action

第五課：運用邏輯表達工具發表課程心得

4P 魅力表達（PREPA 加成）
  1. Power：口語表現（聲音）能力
  2. Picture：將描述視覺化
  3. Passion：傳達真摯的信念
AHA：不只蒐集知識，更要蒐集 AHA
  1. 說法：復述聽到的
  2. 想法：激發更多的
  3. 做法：推導出你的
心得分享 6 步驟
  1. 向大家問候 ＋ 自報家門
  2. 停頓一下，等待回應
  3. 分享前面講者的一個觀點（價值感）
  4. 感謝前面講者帶給你的啟發（意義感）
  5. 接著說自己的觀點（存在感）
  6. 感謝大家的聆聽（情感連線）
最重要的不是說了什麼，而是對方理解了什麼。
