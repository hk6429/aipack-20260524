2026-AIPACK 媒體識讀教案專案

陳乃誠（大乃老師）· 新竹市竹光國中　國文＋資訊組長 · 建立 2026-05-24

專案目標

以方律元媒體識讀三層次（事實 → 立場 → 批判倡議）為理論骨架，產出三個學程（國小 5 下／國中 8 下／高中三）共三份完整教案，符合夢 N AIPACK（原 TPACK）格式。

三層次架構

層次核心能力對應節次
事實層次判斷內容真偽（事實查核）節 1 辨識
立場層次辨識媒體框架與意識形態節 2 思辨
批判層次獨立思考媒體本身的局限性節 3 行動寫作

每節最多 3 小活動，三節共 9 活動。

三份教案總覽

學程主文本工具組合倡議任務
國小 5 下〈懶人包大解密〉＋〈良言一句三冬暖〉Padlet + Canva反假訊息倡議卡（送家人）
國中 8 下翰林 P92-95＋P178-187＋P94-95Padlet + NotebookLM + Canva反假訊息／反霸凌／反刻板印象
高中三〈棘刺刻猴〉＋〈蘋果的滋味〉Padlet + NotebookLM + Canva + CapCut當代蘋果／當代棘刺批判作品

三大數位工具（跨學程通用）

  1. Padlet — 互動討論、立場宣言、作品展
  2. NotebookLM — 同題異源新聞比對（國中以上）
  3. Canva（教育版） — 倡議卡、海報製作
  4. CapCut — 30 秒短影片（高中倡議用）

通用識讀工具

  • 四欄閱讀筆記：源頭 / 內容 / 立場 / 我的判斷
  • 5W 識讀法（Schrock 2009）：Who／What／When／Where／Why
  • 「先看尾巴」網域判讀：.gov／.edu／.org／.com／.idv
  • 韓非子查核邏輯：指其術之原 ＋ 物理可能性

表達架構（依年級遞進）

  • 國小 PR 法（Point-Reason）
  • 國中 PREPA（Point-Reason-Example-Point-Action）
  • 高中 PREPA-A（含進階倡議行動）

參考來源（核心三份）
